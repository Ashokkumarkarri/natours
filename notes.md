# Debugging with `ndb` and Node.js

## 1. Installing `ndb`

- **Globally**:
  ```bash
  npm i ndb --global
  ```
  Installs ndb system-wide.
  As a Dev Dependency (if global installation fails):
  bash
  Copy code
  npm i ndb --save-dev
  Installs ndb locally within the project for development use only.
