---------------------------------------------------------

morgan is a Node.js middleware used for logging HTTP requests.
Why use morgan?: It helps track incoming requests, making debugging easier by providing details like request methods, URLs, status codes, and response times.

How to use morgan?

Install it using npm i morgan.
Import and apply it in your app:

const morgan = require('morgan');
app.use(morgan('dev')); // The 'dev' format logs concise colored outputs
This will automatically log all incoming requests in the specified format.
morgan simplifies the process of monitoring and debugging requests in your Node.js applications.
